PDF生成模板使用说明

模板位置： ~/my-blog/server/templates/pdf-template.html
输出路径： ~/my-blog/server/templates/pdf-template.html

模板特点

  • 极客风格设计： 基于博客的深色主题，保持一致的视觉风格
  • 代码高亮支持： 预定义的代码高亮样式，支持多种编程语言
  • 图片适配优化： 自动调整图片大小，保持清晰度
  • 中文排版优化： 使用Noto Sans SC字体，优化中文阅读体验
  • 打印友好： 专门的打印样式，确保PDF输出质量
  • 响应式设计： 适配不同尺寸的屏幕和纸张

模板变量

模板使用Handlebars风格的变量，需要在服务端进行替换：

// 示例变量对象
const templateData = {
    title: "文章标题",
    author: "作者名",
    date: "2024-01-01",
    category: "技术文章",
    views: "1234",
    content: "<p>文章HTML内容</p>",
    tags: ["JavaScript", "Node.js", "PDF"],
    url: "https://blog.example.com/article/123",
    generatedDate: "2024-01-01 12:00:00"
};

使用示例（Node.js）

const fs = require('fs');
const path = require('path');

// 读取模板
const templatePath = path.join(__dirname, 'templates', 'pdf-template.html');
let template = fs.readFileSync(templatePath, 'utf8');

// 准备数据
const data = {
    title: "深入理解Node.js事件循环",
    author: "极客小明",
    date: "2024-04-05",
    category: "Node.js",
    views: "2567",
    content: "<h2>什么是事件循环</h2><p>事件循环是Node.js的核心机制...</p>",
    tags: ["Node.js", "JavaScript", "异步编程"],
    url: "https://geekblog.com/articles/nodejs-event-loop",
    generatedDate: new Date().toLocaleString('zh-CN')
};

// 简单变量替换（实际应使用模板引擎如Handlebars）
Object.keys(data).forEach(key => {
    const placeholder = `{{${key}}}`;
    const value = Array.isArray(data[key]) ? 
        data[key].map(tag => `<span class=\"tag\">${tag}</span>`).join('') :
        data[key];
    template = template.replace(new RegExp(placeholder, 'g'), value);
});

// 保存生成的HTML
const outputPath = path.join(__dirname, 'generated', 'article-pdf.html');
fs.writeFileSync(outputPath, template);

console.log('PDF模板生成完成！');

与现有样式的集成

模板已经集成了博客的主要样式变量：

  • 颜色方案：深色主题（--bg-primary, --bg-secondary等）
  • 字体：Fira Code + JetBrains Mono + Noto Sans SC
  • 代码样式：与博客一致的代码高亮
  • 间距和排版：保持一致的阅读体验

PDF生成建议

  1. 使用puppeteer或类似工具将HTML转换为PDF
  2. 设置合适的页面尺寸（默认为A4）
  3. 启用打印样式以获得最佳打印效果
  4. 处理图片路径确保PDF中能正确显示图片
  5. 添加页眉页脚（模板已包含基本页脚）
注意： 实际使用时，建议使用模板引擎（如Handlebars、EJS）来处理模板变量替换，而不是简单的字符串替换。
